# GUIA: Como publicar o site da banda 5íncope

## OPÇÃO 1: GitHub Pages (GRATUITO e PROFISSIONAL)

### Passo a passo:
1. Criar conta no GitHub (github.com)
2. Criar um repositório público
3. Fazer upload de todos os arquivos (HTML + pasta imagens)
4. Ativar GitHub Pages nas configurações
5. Seu site ficará em: https://seunome.github.io/5incope

### Vantagens:
✅ Completamente gratuito
✅ SSL automático (https://)
✅ Carregamento rápido mundial
✅ Backup automático
✅ Fácil de atualizar

---

## OPÇÃO 2: Netlify (GRATUITO)

### Passo a passo:
1. Acessar netlify.com
2. Arrastar a pasta do projeto
3. Site online em segundos
4. URL personalizada disponível

### Vantagens:
✅ Deploy instantâneo
✅ URLs personalizadas
✅ Formulários funcionam
✅ CDN global

---

## OPÇÃO 3: Vercel (GRATUITO)

### Passo a passo:
1. Acessar vercel.com
2. Importar projeto
3. Deploy automático

---

## PREPARAÇÃO DO PROJETO:

### Estrutura necessária:
📁 projeto-5incope/
├── 📄 index.html (renomear "Projeto 5íncope.html")
├── 📁 imagens/
│   ├── 📁 logotipo/
│   └── 📁 banda-oficial/
└── 📄 README.md (opcional)

### Ajustes necessários:
1. Renomear arquivo principal para "index.html"
2. Verificar se todos os caminhos das imagens estão corretos
3. Testar localmente antes de publicar

---

## DOMÍNIO PERSONALIZADO (Opcional):

### Opções:
- Comprar domínio (.com, .com.br) - R$ 30-60/ano
- Conectar com GitHub Pages/Netlify/Vercel
- Exemplo: www.5incope.com.br

---

RECOMENDAÇÃO: GitHub Pages é a melhor opção para começar!
